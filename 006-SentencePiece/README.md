Simple example of using SentencePiece, training it on some CommonCrawl data and
tokenizing some existing text:

Examples:

```
king -> ▁|king
ring -> ▁|ring
ringo -> ▁|ring|o
ringo starr -> ▁|ring|o|▁|s|tar|r
erring -> ▁|er|ring
erroring -> ▁|er|ro|ring
terrorizing -> ▁|ter|ro|ri|z|ing
doing -> ▁do|ing
wing -> ▁w|ing
ping -> ▁p|ing
inkling -> ▁|ink|ling
song -> ▁so|ng
bong -> ▁b|ong
nitting -> ▁|n|it|ting
ingo -> ▁in|go
bingo -> ▁b|ing|o
```
